Using Glut with Bloodshed Dev-C++

Download the Glut libraries for Dev-C++ below:

    * glut.h - Save this file to "C:/dev-c++/include/gl/ "
    * glut32.def - Save this file to "C:/dev-c++/lib/ "
    * glut32.dll - Save this file to "C:/windows/command/ "

Note: a copy of the glut32.dll file should accompany any exe files you make using Glut.
This is because it is unlikely that any user that you pass your program on to will have it on their system.

To compile in Dev-C++, Choose Project->Project Options->Parameters->Linker Box
Type " -lopengl32 -lglu32 -lglut32 "
